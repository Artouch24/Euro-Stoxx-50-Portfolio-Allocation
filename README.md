# Euro Stoxx 50 Portfolio Allocation

This project studies whether constrained quantitative portfolio allocation can improve the risk-return profile of an equally weighted portfolio built from the current EURO STOXX 50 constituents.

## Planned strategies

- Equal-weighted portfolio
- Minimum-variance portfolio
- Maximum-Sharpe portfolio
- Comparison with the EURO STOXX 50 benchmark

## Methodology

The project will use daily adjusted prices and will include data cleaning, risk and performance analysis, portfolio optimisation and out-of-sample backtesting.

## Important limitation

The analysis uses the current constituents of the EURO STOXX 50 over the historical period, which introduces survivorship bias.
